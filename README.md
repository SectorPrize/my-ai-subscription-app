# AI Subscription App (React Native + Expo)

Тестовое задание: Приложение с онбордингом, Paywall и сохранением состояния подписки.

### 🛠 Стек технологий:

- **Framework:** React Native (Expo)
- **Language:** TypeScript
- **Navigation:** React Navigation (Stack)
- **Storage:** AsyncStorage (для эмуляции биллинга)

### 🤖 Работа с AI:

В процессе разработки использовался AI для:

1. Генерации базовой структуры навигации.
2. Рефакторинга компонентов (типизация пропсов в TS).
3. Дебаггинга ошибок Metro Bundler и кроссплатформенной совместимости (Web/Mobile).

### 🚀 Как запустить:

1. `npm install`
2. `npx expo start`
